🔧 Subject Migration Tool One-time fix

Your existing subjects were added with the old class-specific system. This tool automatically upgrades them to the new Level Group + Stream model so they appear correctly in score entry dropdowns.

What this does:
• Subjects tagged "SS 1", "SS 2", "SS 3" → converted to SS · All Streams
• Subjects tagged "SS 1 Science", "SS 2 Science" etc → converted to SS · Science
• Subjects tagged "JSS 1", "JSS 2", "JSS 3" → converted to JSS
• Subjects already tagged "ALL" → left unchanged
• Subjects with "dept=ALL" → marked as General (all classes)
